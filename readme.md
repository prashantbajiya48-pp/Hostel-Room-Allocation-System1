# Hostel Room Allocation System

## Project Description
This project is a simple Java application that helps manage hostel room allocation for students. The system stores student details and assigns rooms in an organized way.

## Features
- Add student details
- Allocate room numbers
- Display student records
- Simple console-based interface

## Technologies Used
- Java
- Object Oriented Programming
- ArrayList
- Scanner class

## How to Run the Project
1. Compile the program:
   javac HostelRoomAllocation.java

2. Run the program:
   java HostelRoomAllocation
